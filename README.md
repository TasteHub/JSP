## 프로젝트 소개

치솟는 물가에 외식이 두려워진 요즘, 막상 요리를 시작하기엔 막막한 **자취생** <br>
요리에 관심이 있지만, 레시피를 찾아보긴 귀찮은 **요린이** <br>
이런 사람들의 니즈에 맞춰 기획한 2인 개발 웹사이트입니다. <br>
<br><br>
